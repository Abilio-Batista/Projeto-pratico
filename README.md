# Gerenciador de Contas

Um aplicativo simples feito em Flutter para gerenciar contas a pagar.  
Ele mostra uma lista de contas, permite marcar como pagas e exibe o total que ainda falta pagar.

## Funcionalidades

- Listar contas com título, valor e vencimento  
- Marcar e desmarcar contas como pagas  
- Calcular automaticamente o total das contas não pagas  
- Interface simples e fácil de usar

## Como rodar o projeto

1. Instale o Flutter na sua máquina  
2. Clone ou baixe o projeto  
3. No terminal, entre na pasta do projeto e execute:

flutter run

Estrutura básica:

lib/
 ├── main.dart
 ├── home_page.dart
 ├── models/conta.dart
 └── widgets/conta_card.dart
